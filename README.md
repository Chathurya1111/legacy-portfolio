# Personal Portfolio Website
Java JSP • Servlets • Apache Tomcat

---

## Overview
This is a personal portfolio web application developed using Java JSP, Servlets, HTML, CSS, and Apache Tomcat.

The project was built as a learning exercise to understand Java web development, server-side rendering, and dynamic web application structure using Eclipse IDE.

---

## Features
- Home page with personal introduction
- About page with profile and skills
- Projects showcase section
- Contact page with contact form and social links
- Reusable JSP components (Navbar and Footer)
- Clean multi-page navigation system

---

## Tech Stack

- Backend: Java (JSP, Servlets)
- Frontend: HTML5, CSS3
- Server: Apache Tomcat
- IDE: Eclipse

---

## Project Structure

The project follows a simple MVC-style structure using JSP and Servlets.

- **src/**
  - Contains Java backend code
  - Includes Controller and Model packages
  - Handles application logic and data processing

- **webapp/**
  - Contains all frontend resources
  - Includes JSP pages for UI (index, about, projects, contact, etc.)
  - Contains CSS files for styling

- **WEB-INF/**
  - Contains configuration files (web.xml)
  - Manages application routing and deployment settings

---

## How to Run

1. Install Apache Tomcat (9 or 10)
2. Open Eclipse IDE
3. Import project as Dynamic Web Project
4. Configure Tomcat server in Eclipse
5. Run the project on server
6. Open browser and visit:http://localhost:8080/Portfolio
